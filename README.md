# GAS-bot
[Endoのホームページ](https://endo0507.github.io/created/)で紹介しているbotのソースコード(サンプル)を置いています。<br>
bot_sample.xlsxと、scriptフォルダ内のスクリプトをダウンロードして、サンプルプログラムとして動作させたり、<br>
ソースコードを自由に変更して、OSSとして使うこともできます。
## OSSとして使う場合
このソースコードは、OSIが定義するオープンソースの定義に則って使用可能です。<br>
ソースコードを自由に変更して使うことができます。
## サンプルプログラムとして使う場合
### 最初の準備
1. 自身のGoogleドライブにbot_sample.xlsxをアップロードする
2. bot_sample.xlsxをGoogleスプレッドシートで開き、[ファイル]メニュー→[スプレッドシートとして保存]をする
3. 保存したスプレッドシートを開き、[拡張機能]メニュー→[Apps Script]を開く
4. 左側にある、最初に用意されたCode.gsの[ ⋮ ]をクリックしてリネームするか、ファイルの[＋]アイコン→[スクリプト]を選択
5. scriptフォルダにあるソースコードと同じファイル名を付ける

これ以降の詳しい使い方については、ソースコードの最初にコメントがあります。
## botの紹介
### しずくandまどぐちbot
bot_sample.xlsxの"sizkBot"シートと、sizkBot.gsファイルを使用します<br><br>
[X(\@orugarubot)](https://x.com/orugarubot)で運用しているbotです。<br>
スプレッドシートから投稿内容を取得し、XのAPIにリクエストを送ってポストします。<br>
スプレッドシートから次の投稿時刻を取得し、次の投稿のトリガーを設定します。
### LINE Notifyを活用した「今から帰るよ」bot
bot_sample.xlsxの"時刻表"シートと、LINENotify.gsファイルを使用します<br><br>
Googleフォームで何本後の電車で帰るか回答すると、スプレッドシートからメッセージ内容を取得して送信します。<br>
次の電車の時刻算出、メッセージ内容の作成はスプレッドシート側で出来ているので、スクリプトはただ送信するだけのシンプルなものです。
